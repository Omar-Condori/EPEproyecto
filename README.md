# Perfil de Egreso - Proyecto de Gestión Integral de TI

Proyecto académico de la **Escuela Profesional de Ingeniería de Sistemas** de la **Universidad Peruana Unión**. Corresponde a las competencias del perfil de egreso, desarrollando un **Sistema de Gestión Financiera y Administrativa ('Finanzas-Clinic')** para el caso de estudio **Clínica Médica**, con una arquitectura basada en **Spring Boot + Angular sobre AWS**.

## Estructura del proyecto

### CE01 - Gestión de Tecnologías de Información

| Entregable | Descripción |
|---|---|
| **Entregable 1** | Diagnóstico Organizacional y Alineamiento Estratégico |
| **Entregable 2** | Business Case del Proyecto (Caso de Negocio) |
| **Entregable 3** | Plan de Gestión del Proyecto (PMBOK / Agile) |
| **Entregable 4** | Modelado de Procesos AS-IS / TO-BE |

### CE02 - Ingeniería de Software

| Entregable | Descripción |
|---|---|
| **Entregable 1** | Requerimientos y Diseño del Sistema |
| **Entregable 2** | Plataforma de Datos del Sistema |
| **Presentaciones** | Finanzas-MED: Plataforma de Datos y Presentación del proyecto |

### CE03 - Infraestructura Tecnológica

| Entregable | Descripción |
|---|---|
| **Entregable 1** | Diseño de Red para el Sistema Financiero sobre AWS |
| **Entregable 2** | Planificación de Seguridad para el Sistema Financiero |
| **Entregable 3** | Diseño de Centro de Datos - Arquitectura Cloud Híbrida sobre AWS |

## Tecnologías utilizadas

- **Backend:** Spring Boot
- **Frontend:** Angular
- **Infraestructura:** AWS (Cloud Híbrida)
- **Base de datos:** Según diseño definido en CE022
