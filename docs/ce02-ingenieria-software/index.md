# Ingeniería de Software

## Competencia CE02

Esta área cubre el **ciclo de vida del desarrollo de software**, desde la especificación de requerimientos hasta el diseño de la plataforma de datos, utilizando metodologías ágiles y prácticas de ingeniería de software.

## Entregables

### [Entregable 1: Requerimientos y Diseño del Sistema](entregable-1.md)

Especificación de requerimientos funcionales y no funcionales, prototipo navegable, diseño arquitectónico con diagramas de componentes y despliegue, y diagramas UML (casos de uso, secuencia, estados).

### [Entregable 2: Plataforma de Datos del Sistema](entregable-2.md)

Modelo de datos completo con 6 módulos (seguridad, catálogo, tesorería, logística, cuentas corrientes, facturación), scripts DDL/DML para PostgreSQL, triggers, procedimientos almacenados, consultas analíticas y plan de seguridad.

## Tecnologías

- **Backend:** Spring Boot 4.0
- **Frontend:** Angular
- **Base de datos:** PostgreSQL 16
- **Mensajería asíncrona:** Kafka
- **Metodología:** Ágil (Scrum)
- **Modelado:** UML, BPMN 2.0
