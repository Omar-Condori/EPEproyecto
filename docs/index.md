# Sistema de Gestión Financiera y Administrativa <small>Finanzas-Clinic</small>

Bienvenido al proyecto **Sistema Finanzas - Perfil de Egreso**, un **ERP Financiero** desarrollado como proyecto de aplicación especializada para la **Clínica Médica**, en el marco de las competencias del perfil de egreso de la **Escuela Profesional de Ingeniería de Sistemas** de la **Universidad Peruana Unión**.

---

## 🎯 Objetivo del Proyecto

Diseñar e implementar un **Sistema de Gestión Financiera y Administrativa** que permita a la Clínica Médica automatizar sus procesos de cobro, facturación, control de inventarios y conciliación de convenios, reduciendo ineficiencias operativas y mejorando la toma de decisiones basada en datos.

## 🏗️ Stack Tecnológico

| Componente | Tecnología |
|---|---|
| **Backend** | Spring Boot 4.0 |
| **Frontend** | Angular |
| **Base de datos** | PostgreSQL 16 |
| **Mensajería** | Kafka |
| **Proxy** | Nginx |
| **Infraestructura** | AWS (EC2, RDS, S3, VPC) |
| **Contenedores** | Docker |
